# 🪔 Diwali Sales Data Analysis

## 📌 Project Overview
Exploratory data analysis on a Diwali retail sales dataset to study customer purchasing behavior across demographics and product categories.

## 🛠️ Tech Stack
- Python, Pandas, NumPy, Matplotlib, Seaborn

## 🔍 What I Analyzed
- Gender-wise, age-wise, and state-wise purchasing patterns
- Occupation-wise and product category breakdown
- Grouped aggregations using groupby() for segment insights

## 📊 Key Findings
- Married women aged 26–35 from UP, Maharashtra, and Karnataka were the highest buyers
- Food, Clothing, and Electronics were the top-selling categories
- IT, Healthcare, and Aviation sectors showed the highest spending

## 💡 Business Recommendations
- Target female customers aged 26–35 with personalized offers
- Stock up on Food & Electronics categories before festive season
- Focus marketing spend on top 3 states
